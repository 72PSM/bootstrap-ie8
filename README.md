#Bootstrap 5 for IE8 (Alpha)

Bootstrap 3 drops support for Internet Explorer 8, but you can add it back by simply adding this conditional CSS.

```html
<!--[if lt IE 9]>
    <link href="/css/bootstrap-ie8.css" rel="stylesheet">
<![endif]-->
```
